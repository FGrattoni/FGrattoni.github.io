Evaluation of Regression Model¶

We apply three methods to evaluate our model.

Method 1: Residual Analysis¶

This first method is used to see if there exist some pattern in the residuals (difference between prediction and observed values).

In [7]:
plt.figure(figsize=(5,5))
plt.scatter(y_train_pred, y_train_pred - y_train, c = "steelblue", marker='o', label ="Training data")
plt.scatter(y_test_pred, y_test_pred - y_test, c = "orange", marker='x', label ="Test data")
plt.xlabel("Predicted values")
plt.ylabel("Residuals")
plt.title("Residual analysis\n", fontsize=15)
plt.legend(loc= "upper left")
plt.hlines(y=0, xmin=0, xmax=40, lw=2)
plt.xlim([5,35])
plt.show()
